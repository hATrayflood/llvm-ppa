LLVM/clang PPA
==============

Porting from http://packages.qa.debian.org/c/clang.html to Ubuntu.

provides
--------
    /usr/bin/clang-3.0
    /usr/bin/clang++-3.0

usage
-----
    sudo add-apt-repository ppa:h-rayflood/llvm
    sudo apt-get update
    sudo apt-get install clang-3.0
